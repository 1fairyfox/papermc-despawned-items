PaperMC Despawned Items ships as a single plugin jar for Paper 1.21.x servers on Java 21 (it also loads on 26.x servers via Paper's forward-compatibility). All builds are published on GitHub Releases — the source of truth for every version.

Asset: papermc-despawned-items-<version>.jar — one shaded jar, Kotlin runtime included, no other dependencies. Each release asset carries a GitHub build-provenance attestation.

Install

Drop the jar into your server's plugins/ folder and restart. The plugin registers as papermc-despawned-items and creates its data folder of the same name on first run. Full setup: Tutorials.
